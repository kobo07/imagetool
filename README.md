# 图片处理工具 / Image Processor

一个现代化的静态网页应用，用于图片格式转换、压缩和批量处理。

## ✨ 功能特性

- 📤 **单个/批量上传** - 支持拖拽上传或点击选择多个图片
- ✂️ **图片编辑** - 裁剪、旋转、翻转、调整分辨率
- 🔄 **格式转换** - 支持 JPEG、PNG、WebP、BMP、GIF 格式互转
- 🗜️ **智能压缩** - 可调节压缩质量（1-100%），PNG自动转JPEG压缩
- 🧠 **智能提示** - PNG压缩时显示格式转换提示
- 📊 **实时预览** - 显示压缩前后大小对比和压缩率
- 💾 **批量下载** - 一键打包所有图片为ZIP文件下载

### 音频处理工具
- **格式转换**：支持 MP3、WAV、OGG、M4A、FLAC、AAC、WMA 等格式
- **压缩优化**：可调节比特率（64-320 kbps）
- **批量处理**：同时处理多个音频文件
- **在线编辑**：
  - 波形可视化显示
  - 音频播放和暂停
  - 裁剪音频长度
  - 实时预览编辑效果
- **批量下载**：支持将所有处理后的音频打包为 ZIP 下载

## 技术栈

- **前端框架**：原生 HTML5 + CSS3 + JavaScript
- **图片处理**：Canvas API + Cropper.js
- **音频处理**：Web Audio API
- **文件打包**：JSZip
- **UI 设计**：现代化响应式设计，支持深色/浅色主题切换
## 📋 支持的格式

- **输入格式**: JPG, JPEG, PNG, WebP, GIF, BMP, TIFF, ICO 等所有浏览器支持的图片格式
- **输出格式**: JPEG, PNG, WebP, BMP, GIF

## 🛠️ 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- Canvas API 用于图片处理
- Cropper.js 用于图片编辑
- JSZip 用于打包下载
- 完全静态，可部署到任何 Web 服务器

## 🎯 特色功能

### 图片编辑
- **裁剪功能** - 自由裁剪图片区域
- **旋转翻转** - 左右旋转90度，水平/垂直翻转
- **调整分辨率** - 自定义宽度和高度
- **保持比例** - 可选择保持原始宽高比
- **批量编辑** - 逐个编辑多张图片

### 智能压缩控制
- 滑块调节压缩质量（1-100%）
- PNG图片启用压缩时自动转换为JPEG格式
- 友好的提示信息，让用户了解转换行为
- 实时显示压缩前后大小对比和压缩率

### 批量处理
- 支持同时上传多个图片
- 统一设置应用到所有图片
- 一键打包下载为ZIP压缩包
- 自动生成带时间戳的文件名

### 用户体验
- 拖拽上传支持
- 实时预览
- 流畅的动画效果
- 响应式设计，支持移动设备
- 白天/夜晚主题切换
- 主题偏好自动保存到本地
