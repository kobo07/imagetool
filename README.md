# 图片处理工具 / Image Processor

一个现代化的静态网页应用，用于图片格式转换、压缩和批量处理。

## ✨ 功能特性

- 📤 **单个/批量上传** - 支持拖拽上传或点击选择多个图片
- 🔄 **格式转换** - 支持 JPEG、PNG、WebP、BMP、GIF 格式互转
- 🗜️ **智能压缩** - 可调节压缩质量（1-100%），PNG自动转JPEG压缩
- 🧠 **智能提示** - PNG压缩时显示格式转换提示
- 📊 **实时预览** - 显示压缩前后大小对比和压缩率
- 💾 **批量下载** - 一键打包所有图片为ZIP文件下载
- 🎨 **现代UI** - 深色/浅色主题切换，流畅动画，响应式设计
- 🌓 **主题切换** - 支持白天/夜晚模式，自动保存偏好设置
- 🔒 **隐私保护** - 所有处理在浏览器本地完成，不上传服务器

## 🚀 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 拖拽图片到上传区域或点击"选择图片"按钮
3. 选择输出格式（保持原格式/JPEG/PNG/WebP）
4. 可选：启用压缩并调节压缩质量
5. 点击"开始处理"按钮
6. 查看处理结果并下载图片

## 📋 支持的格式

- **输入格式**: JPG, JPEG, PNG, WebP, GIF, BMP, TIFF, ICO 等所有浏览器支持的图片格式
- **输出格式**: JPEG, PNG, WebP, BMP, GIF

## 🛠️ 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- Canvas API 用于图片处理
- JSZip 用于打包下载
- 完全静态，可部署到任何 Web 服务器

## 🎯 特色功能

### 智能压缩控制
- 滑块调节压缩质量（1-100%）
- PNG图片启用压缩时自动转换为JPEG格式
- 友好的提示信息，让用户了解转换行为
- 实时显示压缩前后大小对比和压缩率

### 批量处理
- 支持同时上传多个图片
- 统一设置应用到所有图片
- 一键打包下载为ZIP压缩包
- 自动生成带时间戳的文件名

### 用户体验
- 拖拽上传支持
- 实时预览
- 流畅的动画效果
- 响应式设计，支持移动设备
- 白天/夜晚主题切换
- 主题偏好自动保存到本地
